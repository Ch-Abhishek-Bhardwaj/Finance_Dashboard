# 💰 Finance Dashboard (FinTrack)

FinTrack is built to simplify financial management. Whether you are tracking daily expenses, monitoring a growing investment portfolio, or setting monthly budgets, this dashboard consolidates all your financial data into one easy-to-read hub.

🔗 **Live Demo:** https://dashboard-fintrack.netlify.app/

---

## ✨ Key Features

- 📊 **Interactive Data Visualizations**  
  Dynamic charts and graphs for income vs. expenses, category breakdowns, and net worth trends over time.

- 💰 **Expense & Income Tracking**  
  Easily log transactions, categorize them, and view historical data.

- 📈 **Investment Portfolio**  
  Track stocks, mutual funds, or crypto assets with updated market value.

- 📱 **Responsive Design**  
  Fully optimized for desktop, tablet, and mobile.

---

## 🛠️ Tech Stack

- **Frontend:** React.js, CSS, Recharts / Chart.js
